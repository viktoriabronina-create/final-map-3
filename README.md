# Анализ пешеходной доступности станций метрополитена района Южное Бутово (г. Москва)

Главная задумка Проекта 3 - создание интерактивной ГИС-карты, на которой отображаются зоны нормативной пешеходной доступности (5, 10, 15 минут) от выходов метро.

На карте оотображены следующие тематические слои: 
- типы землепользования (для понимания какие именно городские территории не оснащены пешеходной доступностью), 2
- застройка района (с возможностью получения информации по типу здания - всплывающая подсказка)
- улично-дорожная сеть (исключительно пешеходная, так как на карте определяется пешеходная доступность)
- обслуживающие станции метрополитена
- доступность станций метрополитена (построение изохрон по графу пешеходной сети)/
