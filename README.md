# Анализ пешеходной доступности станций метрополитена района Южное Бутово (г. Москва)

Главная задумка проекта - создание интерактивной ГИС-карты, на которой отображаются зоны нормативной пешеходной доступности (5, 10, 15 минут) от выходов метро в районе Южное Бутово. 
Такое представление норматива позволяет оценить пешеходную доступность района в целом.(🥲)

На карте отображены следующие тематические слои: 
- типы землепользования (для оценки городских территорий, которые расположены вне нормативных зон пешеходной доступности)
- застройка района (с возможностью получения информации по типу здания - "всплывающая" подсказка)
- улично-дорожная сеть (исключительно пешеходная, так как суть карты в отображении пешеходной доступности)
- обслуживающие станции метрополитена
- доступность станций метрополитена (построение изохрон по графу пешеходной сети)
